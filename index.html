SGEX Workbench

WHO SMART Guidelines Exchange

A browser-based, standards-compliant collaborative editor for WHO SMART Guidelines Digital Adaptation Kits (DAKs).

This workbench provides a collaborative environment for editing WHO SMART Guidelines content stored in GitHub repositories, using JSON Forms for standards compliance and accessibility.
